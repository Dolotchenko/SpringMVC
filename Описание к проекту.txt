Проект написан с использованием Spring MVC. Так же использовался Hibernate. Посредством Hibernate происходит взаимодействие с БД.
В качестве Сервера приложений используется TomCat.
Название проекта "Выдача кредита в Банке". Приложение представляет собой упрощенную бизнес-логику процесса выдачи займа клиенту.
Существует 3 типа пользователей. Каждому пользователю предоставлены особые права доступа и перечислен список доступных
страниц для отображения. В качестве авторизации на странице использован Spring Security. настройки в файле security.xml
В качестве отображения данных используется шаблонизатор FreeMarker.
В приложении продемонстрирована работа с REST сервисом. Описание в классе "MyRestController" Данные на страницу отправляются ч/з Json. Работа на странице происходит
через ajax запрос.
Так же приведен пример сервиса, который описан в классе "MyService". Он является "прослойкой" между моделью и контроллером. 
В нем должна храниться логика взаимодествия между моделью и контроллером. Это сделано для того, чтобы при "переезде"
на другую БД не пришлось переписывать логику взаимодействия с БД.
